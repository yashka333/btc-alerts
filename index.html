Telegram-уведомления

Как получить Chat ID: 1) в Telegram откройте бота @userinfobot и нажмите Start — он пришлёт в ответ ваш личный Id, это и есть Chat ID для личных сообщений. 2) обязательно также откройте бота @ImoAndVladiAlert_bot и нажмите Start (иначе он не сможет писать вам, пока вы не начали с ним диалог). 3) для группы: добавьте @ImoAndVladiAlert_bot в группу, затем временно добавьте туда @RawDataBot — он пришлёт id группы (число со знаком минус), после чего его можно удалить из группы.
